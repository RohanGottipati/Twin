## ROI and value case

An intervention value case must keep four categories separate:

1. Measured inputs, with source and observation date.
2. Modeled monetized benefits, with method and uncertainty.
3. Unvalidated assumptions, including demand and delivery risk.
4. Scenario ranges, including downside and upside sensitivity.

Use `ROI = (validated monetized benefits - lifecycle costs) / lifecycle costs`
only when lifecycle costs and monetized benefits are supported. Lifecycle cost
includes capital, operating, maintenance, renewal, financing, and material
disruption costs over the analysis horizon. Report NPV, benefit-cost ratio,
payback period, discount rate, horizon, and sensitivity when available. If an
input is missing, report the validation need and do not invent a return.
